# Olist-Brazillian-E-Commerce-data-Analysis
This project focuses on analyzing sales, customers, deliveries and reviews data using Python, SQL, and Power BI.

This project analyzes 100K+ orders, 99K+ customers, and 3K+ sellers to uncover trends in sales performance, customer behavior, delivery efficiency, and revenue growth.

Key Highlights : 
 R$16M+ total revenue analyzed across 2016–2018
 Identified peak sales periods (Q1 2018) and monthly growth trends
 Discovered that ~97% customers are one-time buyers → huge retention opportunity
 Analyzed delivery performance → 7,800+ late deliveries impacting customer satisfaction
 Found 74% transactions via credit card → opportunity for targeted offers
 Identified top-performing products, categories & sellers
